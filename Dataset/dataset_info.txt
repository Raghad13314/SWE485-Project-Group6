Original Purpose:
-----------------
PhiUSIIL Phishing URL Dataset is a substantial dataset comprising 134,850
legitimate and 100,945 phishing URLs. Most of the URLs analyzed, while
constructing the dataset, are the latest URLs. Features are extracted from
the source code of the webpage and URL. Features such as CharContinuationRate,
URLTitleMatchScore, URLCharProb, and TLDLegitimateProb are derived from
existing features.

Dataset Summary:
-----------------
- Number of Instances : 235,795 (134,850 legitimate + 100,945 phishing)
- Number of Features  : 54
- Feature Types       : Real, Categorical, Integer
- Subject Area        : Computer Science
- Associated Task     : Classification
- Target Variable     : label (1 = Legitimate, 0 = Phishing)
- Missing Values      : No
